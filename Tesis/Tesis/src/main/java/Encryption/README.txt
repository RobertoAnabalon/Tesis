BETA Encrypted Storage.
Implementacion de cifrado AES y RSA.


Se puede implementar mas algoritmos si se desea.
Implementable en cualquier software debido a que está en java, realizando ajustes necesarios al código.
Implementado y semi funcional en Springboot, realizando ajustes en el config y en bean.
AlgoritmoC.java es para dejar en claro que se pueden implementar aun mas algoritmos.
